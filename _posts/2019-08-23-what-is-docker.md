### рЃарЃљ рЃљрЃарЃўрЃА┬арЃЊрЃЮрЃЎрЃћрЃарЃў?

  

рЃерЃћрЃю рЃљрЃџрЃЉрЃљрЃЌ рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃњрЃўрЃњрЃўрЃљ рЃЊрЃЮрЃЎрЃћрЃарЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃЊрЃљ рЃњрЃўрЃцрЃўрЃЦрЃарЃўрЃљ, рЃЌрЃБ рЃарЃљрЃерЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃЮрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃњрЃћрЃДрЃћрЃюрЃћрЃЉрЃўрЃюрЃљ рЃўрЃА. рЃљрЃЏ рЃАрЃбрЃљрЃбрЃўрЃљрЃерЃў, рЃАрЃгрЃЮрЃарЃћрЃЊ рЃљрЃЏ рЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃќрЃћ рЃњрЃљрЃњрЃфрЃћрЃЏрЃЌ рЃърЃљрЃАрЃБрЃ«рЃА.

рЃЊрЃЮрЃЎрЃћрЃарЃў рЃљрЃарЃўрЃА **рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА** рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ (Runtime environmet), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃФрЃџрЃћрЃЋрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА, рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃўрЃќрЃЮрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃарЃћрЃЏрЃЮ, рЃАрЃљрЃЊрЃљрЃф рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃАрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃЌрЃљрЃЋрЃА рЃЏрЃЮрЃБрЃДрЃарЃў рЃЏрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА(**dependencies**), рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃЊрЃљ рЃљ.рЃе. рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃАрЃљрЃГрЃўрЃарЃЮрЃљ рЃљрЃдрЃюрЃўрЃерЃюрЃБрЃџрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ.

рЃЊрЃЮрЃЎрЃћрЃарЃў рЃљрЃАрЃћрЃЋрЃћ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ рЃЏрЃљрЃерЃўрЃю, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃњрЃўрЃюрЃЊрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃБрЃЎрЃЋрЃћ рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў, рЃарЃљрЃЏрЃљрЃф рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃгрЃЋрЃўрЃЮрЃА рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ рЃњрЃљрЃЦрЃЋрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃў, рЃАрЃљрЃЊрЃљрЃф рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃўрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **Node 6** рЃЋрЃћрЃарЃАрЃўрЃљрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃњрЃўрЃюрЃЊрЃљ рЃњрЃљрЃБрЃерЃЋрЃљ рЃљрЃ«рЃљрЃџрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ - рЃўрЃњрЃўрЃЋрЃћ рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ, **Node 12** рЃЋрЃћрЃарЃАрЃўрЃўрЃЌ. рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃћрЃЦрЃюрЃћрЃЉрЃљ рЃўрЃќрЃЮрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃарЃћрЃЏрЃЮ, рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃљрЃЊ рЃАрЃ«рЃЋрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ **Node**\-рЃўрЃА рЃЋрЃћрЃарЃАрЃўрЃљрЃф, рЃарЃљрЃф рЃљрЃа рЃЊрЃљрЃБрЃерЃЋрЃћрЃЉрЃА рЃЋрЃћрЃарЃАрЃўрЃљрЃЌрЃљ рЃЎрЃЮрЃюрЃцрЃџрЃўрЃЦрЃбрЃА.

### рЃарЃљ рЃљрЃарЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃў?

![](img/1__bOrphmm1BtsoUc9rND7lmw.png)

рЃарЃЮрЃњрЃЮрЃарЃф рЃБрЃЎрЃЋрЃћ рЃљрЃЋрЃдрЃюрЃўрЃерЃюрЃћрЃЌ, рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃў рЃљрЃарЃўрЃА рЃўрЃќрЃЮрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃљрЃарЃћрЃЏрЃЮ, рЃарЃЮрЃЏрЃџрЃўрЃАрЃЌрЃЋрЃўрЃАрЃљрЃф рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃў рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃА (**Network, hardware┬аРђд**)

рЃерЃћрЃАрЃљрЃФрЃџрЃЮрЃљ рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљрЃќрЃћ рЃўрЃДрЃЮрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў рЃЉрЃћрЃЋрЃарЃў рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃў, рЃАрЃ«рЃЋрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃЌ. рЃарЃЮрЃЏрЃћрЃџрЃЌрЃљ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃљрЃа рЃљрЃўрЃАрЃљрЃ«рЃћрЃЉрЃљ рЃАрЃ«рЃЋрЃљ рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃћрЃЉрЃќрЃћ.

### **рЃЊрЃЮрЃЎрЃћрЃарЃў рЃљрЃа рЃљрЃарЃўрЃА рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃЏрЃљрЃюрЃЦрЃљрЃюрЃљ┬а!**

  

![](img/1__8rLp2Zg1d3ablZg0MOPQag.png)

рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃЏрЃљрЃюрЃЦрЃљрЃюрЃўрЃАрЃњрЃљрЃю рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃўрЃЌ, рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃў, рЃљрЃа рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃА рЃЏрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃБрЃџ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЎрЃћрЃарЃюрЃћрЃџрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА рЃЊрЃљ рЃњрЃљрЃљрЃЕрЃюрЃўрЃљ рЃАрЃљрЃЎрЃБрЃЌрЃљрЃарЃў **_namespace_** (рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃўрЃџрЃў рЃарЃћрЃАрЃБрЃарЃў) рЃарЃљрЃф рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃўрЃЏрЃљрЃА, рЃарЃЮрЃЏ **Network**, **Memory** рЃЊрЃљ рЃљ.рЃе.рЃњрЃљрЃЏрЃЮрЃДрЃЮрЃцрЃА рЃърЃљрЃбрЃљрЃарЃљ рЃюрЃљрЃгрЃўрЃџрЃА рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА.

**docker build**\- рЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃљрЃЊ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃфрЃћрЃЌ **\-t** (**tag**), рЃарЃљрЃф рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЕрЃЋрЃћрЃюрЃў **Image**\-рЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў, рЃЌрЃБ рЃљрЃа рЃњрЃљрЃЊрЃљрЃЋрЃфрЃћрЃЏрЃЌ **tag name** -рЃА рЃЊрЃЮрЃЎрЃћрЃарЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃЏрЃўрЃљрЃюрЃўрЃГрЃћрЃЉрЃА рЃарЃћрЃюрЃЊрЃЮрЃЏ **tag name**\-рЃА.

рЃљрЃАрЃћрЃЋрЃћ, рЃЉрЃЮрЃџрЃЮрЃерЃў рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃџрЃљрЃЊ рЃЋрЃгрЃћрЃарЃЌ рЃгрЃћрЃарЃбрЃўрЃџрЃА(**.**)-рЃА рЃарЃљрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃА..

![](img/1__YTR1Jjs2RnimDAzR5NQ5Ww.png)

рЃЌрЃБ рЃњрЃљрЃЋрЃБрЃерЃЋрЃћрЃЉрЃЌ **Command**\-рЃА

docker images

┬арЃЋрЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃарЃЮрЃЏ рЃерЃћрЃўрЃЦрЃЏрЃюрЃљ рЃЮрЃарЃў рЃљрЃ«рЃљрЃџрЃў **image** - **myapp** рЃЊрЃљ **node**(рЃарЃљрЃЊрЃњрЃљрЃю рЃБрЃЎрЃЋрЃћ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЦрЃЮрЃюрЃЊрЃљ, рЃЦрЃћрЃерЃўрЃЊрЃљрЃю рЃљрЃўрЃдрЃЮ)

рЃўрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃў рЃњрЃљрЃЋрЃБрЃерЃЋрЃљрЃЌ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ **Command**\-рЃА:

docker run myapp **#**(рЃАрЃљрЃЊрЃљрЃф myapp рЃЌрЃЦрЃЋрЃћрЃюрЃў image-рЃА рЃАрЃљрЃ«рЃћрЃџрЃўрЃљ).

![](img/1__u__iO6vnzpL2ODOpp805nHw.png)

рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃћрЃЉрЃўрЃА рЃАрЃљрЃюрЃљрЃ«рЃљрЃЋрЃљрЃЊ рЃЋрЃБрЃерЃЋрЃћрЃЉрЃЌ **Command**\-рЃА:

docker ps

рЃЌрЃБ рЃЕрЃЋрЃћрЃю рЃерЃћрЃЋрЃћрЃфрЃЊрЃћрЃЉрЃўрЃЌ **localhost:8000**\-рЃќрЃћ рЃерЃћрЃАрЃЋрЃџрЃљрЃА рЃњрЃЋрЃўрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА **Network error**\-рЃА рЃарЃљрЃЊрЃњрЃљрЃю рЃарЃЮрЃњрЃЮрЃарЃф рЃБрЃЎрЃЋрЃћ рЃЋрЃљрЃ«рЃАрЃћрЃюрЃћрЃЌ, рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃў рЃљрЃарЃўрЃА рЃўрЃќрЃЮрЃџрЃўрЃарЃћрЃЉрЃБрЃџ рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў.

![](img/1__ToIhNQWopIDnXdgsfcHgew.png)

рЃўрЃЏрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏ рЃЊрЃљрЃЋрЃљрЃЏрЃДрЃљрЃарЃЮрЃЌ рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃЌрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЋрЃ«рЃАрЃюрЃљрЃЌ рЃърЃЮрЃарЃбрЃў.

рЃарЃўрЃАрЃЌрЃЋрЃўрЃАрЃљрЃф **docker run** рЃА рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЊрЃљрЃЋрЃфрЃћрЃЌ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў **\-p 8000:8000**

рЃарЃљрЃф рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃўрЃЏрЃљрЃА, рЃарЃЮрЃЏ **8000** рЃърЃЮрЃарЃбрЃўрЃЊрЃљрЃю рЃЋрЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃўрЃЌ рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃўрЃА **8000** рЃърЃЮрЃарЃбрЃА.

docker run -p 8000:8000 myapp

![](img/1__Jnh0RyutyVPVnCuqxAGe0A.png)

рЃърЃЮрЃарЃбрЃўрЃА рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃљрЃЋрЃўрЃюрЃљрЃ«рЃљрЃЋрЃЌ, рЃарЃЮрЃЏ рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃА рЃЏрЃўрЃћрЃюрЃўрЃГрЃљ рЃърЃЮрЃарЃбрЃў рЃЊрЃљ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЕрЃЋрЃћрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃњрЃљрЃЋрЃ«рЃАрЃюрЃљрЃЌ.

![](img/1__tRvPIyp3RNgdX__rbkVCQ0w.png)

  

### рЃЊрЃљ рЃЏрЃљрЃўрЃюрЃф, рЃарЃљрЃбрЃЮрЃЏ рЃЊрЃЮрЃЎрЃћрЃарЃў?┬а­Ъцћ

  

рЃЊрЃЮрЃЎрЃћрЃарЃЏрЃљ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃб рЃърЃарЃЮрЃфрЃћрЃАрЃћрЃЉрЃерЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃЉрЃћрЃЋрЃарЃў рЃарЃљрЃЏ рЃерЃћрЃфрЃЋрЃљрЃџрЃљ, рЃЏрЃљрЃю рЃЉрЃћрЃЋрЃарЃљрЃЊ рЃБрЃцрЃарЃЮ рЃњрЃљрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃљ рЃЊрЃћрЃърЃџрЃЮрЃўрЃЏрЃћрЃюрЃбрЃў.┬а

рЃЊрЃдрЃћрЃАрЃЊрЃдрЃћрЃЮрЃЉрЃўрЃЌ рЃФрЃљрЃџрЃўрЃљрЃю рЃЉрЃћрЃЋрЃарЃў рЃАрЃћрЃарЃЋрЃўрЃАрЃў (**AWS ECS, AWS EKS, AWS Elastic beanstallk, GKE** рЃЊрЃљ рЃљ.рЃе.)рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃерЃўрЃф рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃљрЃЋрЃбрЃЋрЃўрЃарЃЌрЃЮрЃЌ, рЃЕрЃЋрЃћрЃюрЃў рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃћрЃЉрЃў рЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЏрЃЮрЃЋрЃљрЃњрЃЋрЃљрЃарЃЮрЃЌ рЃўрЃАрЃћрЃЌрЃў рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ рЃАрЃЦрЃћрЃўрЃџрЃўрЃюрЃњрЃў.┬а

рЃарЃљрЃф рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃўрЃЏрЃљрЃА, рЃарЃЮрЃЏ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃќрЃарЃЊрЃљрЃАрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃЋрЃАрЃбрЃљрЃарЃбрЃЮрЃЌ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃћрЃЉрЃў.
